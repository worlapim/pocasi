Toto je jednoduchá webová aplikace pro zobrazení předpovědi počasí, vytvořena pomocí Visual Studio. 
Aplikace komunikuje přes API se službou OpenWeatherApi a zobrozuje tyto data uživateli.

Aplikace byla testována v těchto prohlížečích
 - Microsoft Edge
 - Google Chrome
 - Firefox
 - Brave
 - Avast Secure Browser

 V třídě Constants je použit můj soukromý klíč z WeatherAPI k mému zdarma účtu, který jsem dal deaktivovat. 
 Pro zprovoznění si založte vlastní účet (klidně zdarma) a svůj vygenerovaný klíč vložte do konstanty OPEN_WEATHER_API_KEY.
